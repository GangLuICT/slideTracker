Installing the PowerPoint Add-in

STEP 1:
Make sure you have Windows and PowerPoint 2010/13 installed on your system. Then download the installer:
Download
STEP 2:
Make sure all MS office programs are closed.
STEP 3:
Unizip the file and run setup.exe. (It might be necessary to click OK to trust the program.)
STEP 4:
Finish the installation and run PowerPoint.
STEP 5:
Go to the add-in tab and verify you can see slideTracker. Done!

Using the add-in:

STEP 1:
Go to the add-in tab and adjust you settings (code-banner display location, etc.).
STEP 2:
Hit 'export' and wait for the slides to be uploaded to the slideTracker server.
STEP 3:
Once the upload is finished we will take care of the rest. Simply start presenting like usual. Closing PowerPoint will take care of cleaning up.
